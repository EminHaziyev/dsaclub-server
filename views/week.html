Arrays & Strings

Master the fundamentals of arrays and string manipulation. This week covers basic operations, two-pointer techniques, sliding window patterns, and essential problem-solving strategies that form the foundation of algorithmic thinking.

Learning Objectives

  • Understand array data structure and its operations (insertion, deletion, traversal)
  • Master string manipulation techniques and common patterns
  • Learn two-pointer technique for efficient array/string problems
  • Implement sliding window pattern for subarray problems
  • Practice time and space complexity analysis

Key Concepts

1. Array Basics

Arrays are contiguous memory locations that store elements of the same type. Understanding indexing, bounds checking, and basic operations is crucial.

// Array declaration and initialization
int[] arr = {1, 2, 3, 4, 5};

// Accessing elements
int first = arr[0];  // O(1) time complexity

// Modifying elements
arr[2] = 10;  // O(1) time complexity

2. Two-Pointer Technique

Use two pointers moving towards each other or in the same direction to solve problems efficiently in O(n) time.

// Two pointers from both ends
int left = 0, right = arr.length - 1;
while (left < right) {
    // Process elements at left and right
    if (condition) {
        left++;
    } else {
        right--;
    }
}

3. Sliding Window

Maintain a window of elements and slide it across the array to find optimal subarrays.

// Fixed-size sliding window
int windowSum = 0;
for (int i = 0; i < k; i++) {
    windowSum += arr[i];
}

int maxSum = windowSum;
for (int i = k; i < arr.length; i++) {
    windowSum = windowSum - arr[i-k] + arr[i];
    maxSum = Math.max(maxSum, windowSum);
}
