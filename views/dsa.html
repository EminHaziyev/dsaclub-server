Weekly Tasks & Assignments

Practice problems and assignments to reinforce your learning

Week 1: Array Fundamentals

Tasks to Complete:

  • 1. Two Sum - Find indices of two numbers that add up to target
  • 2. Maximum Subarray - Find contiguous subarray with largest sum
  • 3. Rotate Array - Rotate array to the right by k steps
  • 4. Contains Duplicate - Check if array contains duplicates
  • 5. Best Time to Buy Stock - Find maximum profit from stock prices

Deadline: End of Week 1 | Estimated Time: 4-6 hours

✓
Easy

Week 2: Linked List Mastery

Tasks to Complete:

  • 1. Reverse Linked List - Reverse a singly linked list
  • 2. Merge Two Sorted Lists - Merge two sorted linked lists
  • 3. Linked List Cycle - Detect if linked list has a cycle
  • 4. Remove Nth Node - Remove nth node from end of list
  • 5. Middle of Linked List - Find middle node of linked list

Deadline: End of Week 2 | Estimated Time: 5-7 hours

!
Easy

Week 3: Stack & Queue Challenges

Tasks to Complete:

  • 1. Valid Parentheses - Check if parentheses are properly matched
  • 2. Min Stack - Design stack with min element retrieval
  • 3. Queue using Stacks - Implement queue using two stacks
  • 4. Daily Temperatures - Find next warmer temperature
  • 5. Evaluate RPN - Evaluate reverse polish notation

Deadline: End of Week 3 | Estimated Time: 6-8 hours

!
Medium

Week 4: Tree Traversal Practice

Tasks to Complete:

  • 1. Inorder Traversal - Traverse binary tree inorder
  • 2. Maximum Depth - Find maximum depth of binary tree
  • 3. Same Tree - Check if two trees are identical
  • 4. Symmetric Tree - Check if tree is symmetric
  • 5. Level Order Traversal - Traverse tree level by level

Deadline: End of Week 4 | Estimated Time: 7-9 hours

!
Medium

Week 5: BST Operations

Tasks to Complete:

  • 1. Validate BST - Check if tree is valid BST
  • 2. Insert into BST - Insert value into BST
  • 3. Delete Node in BST - Delete node from BST
  • 4. LCA in BST - Find lowest common ancestor
  • 5. Kth Smallest in BST - Find kth smallest element

Deadline: End of Week 5 | Estimated Time: 8-10 hours

!
Hard

Week 6: Graph Algorithms

Tasks to Complete:

  • 1. Number of Islands - Count islands using DFS/BFS
  • 2. Clone Graph - Deep clone an undirected graph
  • 3. Course Schedule - Detect cycle in directed graph
  • 4. Pacific Atlantic Water Flow - Multi-source BFS problem
  • 5. Word Ladder - Shortest path in unweighted graph

Deadline: End of Week 6 | Estimated Time: 10-12 hours

About DSA Club

Learn more about our mission, community, and the team behind the club

🎯 Our Mission

To create a supportive learning environment where students can master data structures and algorithms through collaborative learning, structured curriculum, and hands-on practice. We believe that everyone can excel in DSA with the right guidance and consistent practice.

👥 Community

Join a vibrant community of 200+ students from various departments. Our members range from beginners taking their first steps in programming to advanced students preparing for technical interviews at top tech companies.

📈 Success Stories

Over 150 students have successfully completed our program, with 85% landing internships at major tech companies. Our alumni work at Google, Microsoft, Amazon, Meta, and many other leading organizations.

🏆 What Makes Us Different

Structured Learning Path: 6-week progressive curriculum designed by industry experts
Peer Learning: Study groups, code reviews, and collaborative problem-solving sessions
Industry Mentorship: Guest lectures from software engineers at top tech companies
Interview Preparation: Mock interviews, system design sessions, and behavioral interview practice
Flexible Schedule: Self-paced learning with weekly check-ins and optional study sessions
